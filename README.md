В качестве проекта решил посмотреть, как себя оправдает идея, заключающаяся в том, чтобы модель предсказывала все свойства атома сразу. Тогда модель по идее должна строить полноценное представление об атоме внутри себа, которое в последуюещм будет по разному отображаться выходными весами в различные свойста. То есть по сути в файле mpnn_multitask.py реализована модель, которая выдает вместо одного скалярного свойства атома, сразу все десять. Также некоторые внутренние слои сети были расширены, чтобы повысить ее сложность. В итоге на отобранном датасете из 10ти тысяч атомов эта сетка показала лучший результат по MAE метрике, чем обычная mpnn модель, заточенная под предсказание только диполя. Метрика MAE на валидации - 0.0697 против 0.1362. Однако стоит признать, что в случае обычного mpnn пока что не было реализовано средств для борьбы с переобучением. Графики хода обучения представлены на скриншотах mpnn_multitask.png и mpnn.png.
